# La Baronesa

A single-page event landing page for the La Baronesa winery gala.

## Repository

GitHub: https://github.com/SaeSaeeda/labaronesa.git

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

For example:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080/index.html`.
